### Chrome

- `cmd + alt + arrow` (change between tabs)
- `cmd + shift + i` (open inspector)
- `cmd + r` (reload page)
- `cmd + l` (focus on navigation bar)
- `cmd + n` (open new window)
- `cmd + t` (open new tab)
- `cmd + w` (close current tab)
- `cmd + shit + t` (open closed tab)
- `cmd + q` (close chrome)
